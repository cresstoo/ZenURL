ZenURL Animation
---
An animated mock-up of URL shortener, made by Keynote. 想搭建一个自己的短网址服务，url生成的交互想用翻转效果，用Keynote快速制作出了高保真原型动画。模拟了网址输入、鼠标hover、文字和输入框翻转，生成的gif帧率有点快，keynote源文件要清楚些。
